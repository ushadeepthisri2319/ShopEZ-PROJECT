# 🛒 ShopEZ – MERN Stack E-Commerce Application

## 📖 Overview
ShopEZ is a full-stack e-commerce web application developed using the MERN stack. It provides a seamless online shopping experience with secure user authentication, product browsing, cart management, order placement, and an admin dashboard for product management.

## ✨ Features
- User Registration and Login
- JWT-based Authentication
- Product Browsing
- Shopping Cart Management
- Order Placement
- User Profile Management
- Admin Dashboard
- Responsive User Interface

## 🛠️ Technology Stack
- Frontend: React.js
- Backend: Node.js & Express.js
- Database: MongoDB Atlas
- Authentication: JWT
- Version Control: Git & GitHub
